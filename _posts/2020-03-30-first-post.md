This post is 3 months late. It has been on my mind since January but I never
actually got down to writing it. No excuses though!

As usual, the First Post of 2020 would cover two things. First, [2019 Books]({%
post_url 2019-02-08-2019-books %}). Second, [First Post of 2019]({% post_url
2019-02-08-first-post %}).

The absolute number of books I'm reading has declined steadily over the years.
But that is explained by the more technical books I'm reading, which I do not
post about here. Some examples are "Effective Java", "Designing Data Intensive
Applications", "Cassandra", "The Algorithm Design Manual", etc. This is in line
with what I set out to do in 2019. Get better at algorithms! I like my current
pace of reading and the goal really isn't to brute force 50 books a year. That
would be nice - and perhaps I can get there some day.

For goals in 2019, I set two. I only met the first. I did code more. That can be
seen from LeetCode and HackerRank and the other books I'm reading. For 2020, I'm
going to be keeping this up.

For the second goal, I did not meet it at all. I won't set this as a goal for
2020 but rather something to achieve if the stars align.

I'll keep my goals simple in 2020:

1. Crush it on LeetCode. Solve 500 questions by 1st January 2021.
2. Read 5 general knowledge books.
3. Complete an MIT MicroMasters.
4. And if there's still time, build something on the side.
